```poem
Add a single poem to the information superhighway
A drop of colour
From the tip of your tongue
Into a fast moving river
Will anybody see it
Be enveloped
Merged
A wink in time
I doubt it
But maybe
Some gormless soul
Sitting with fluorescent face
Some highway sitter
Will spot it
```
